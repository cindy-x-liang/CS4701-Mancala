To install Python dependencies:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

To run server:

```bash
python3 -m mancalapi.api
```

To install Node dependencies:

```bash
cd mancalui
npm install
```

To run front-end:

```bash
cd mancalui
npm run dev --open
```
